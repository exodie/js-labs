# Помощник

## Запуск

Запустить `.js` можно с помощью команды

```bash
node main.js
```

\*Если вы уже находитесь в консоли, но ваш путь = `js-labs/`, то выполните:

```bash
node 6/6-1.js
node 6/6-2.js
node 6/6-3.js
```
